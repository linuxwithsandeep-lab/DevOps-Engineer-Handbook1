# Chapter Overview

Cloud Computing has completely transformed the IT industry.

Today, almost every modern company uses cloud platforms instead of purchasing expensive physical servers.

Whether it is:

- Netflix
- Amazon
- Flipkart
- Google
- Microsoft
- Swiggy
- Zomato
- Uber

all rely on cloud infrastructure.

For a DevOps Engineer, learning Cloud is no longer optional.

It is one of the most important skills required in every DevOps job.
